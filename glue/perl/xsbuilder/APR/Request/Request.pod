=head1 NAME

  APR::Request




=head1 SYNOPSIS

  use APR::Request;
  my $req = APR::Request->handle($pool, ...);




=head1 DESCRIPTION

The C<< APR::Request >> module provides the base methods
for interfacing with libapreq2's module api.  It also provides
a few utility functions and constants.
